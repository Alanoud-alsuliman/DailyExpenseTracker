\# DailyExpenseTracker



DailyExpenseTracker is a simple Java console application that helps users record, classify, and manage daily expenses.



\## Project Purpose



The main purpose of this project is to practice software configuration management using Git and GitHub.



\## Basic Features



\- Add daily expenses

\- View all expenses

\- Calculate total expenses

\- Show category report

\- Manage work using branches, commits, and pull requests



\## Team Workflow



Each team member will work on a separate branch and complete a specific task. After finishing the task, the branch will be pushed to GitHub and merged into the main branch using a Pull Request.
