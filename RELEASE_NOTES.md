\# Release Notes

\## v1.0 - Initial Release



\### Summary

The first version of the project includes the basic Java application structure and the main expense management features.



\### Changes

\- Added the basic Java project structure.

\- Added the main menu for the Daily Expense Tracker.

\- Added the Expense class.

\- Added the ability to add expenses.

\- Added the ability to view recorded expenses.
